Notes App

Front end Development / User Experience / Web Interface

a note-taking application with React.js, featuring a markdown editor for users to easily save, delete, and manage their notes. With the integration of local storage, the app ensures that users' notes persist across sessions, even after closing and reopening the browser. Finance Tracker

Full stack Development / User Authentication / Real Time Data Management

Finance Tracker App using React for the frontend and Firebase for backend services. It features Firebase Authentication, allowing users to securely log in and manage their financial transactions. Each transaction is saved to Firebase Collections, ensuring data is user-specific and readily available upon login.

About me

My name is Michael, a dedicated and innovative software engineer with a solid grasp of full-stack development, combining React and TypeScript for front-end solutions, and Java for robust backend systems. My recent internship at Amazon sharpened my abilities in creating scalable and maintainable software that powers user-centric experiences.

I take pride in my proficiency in JavaScript and Python, which complements my front-end and back-end development skills. I’ve translated these technical capabilities into tangible projects that not only meet user needs but are also built for long-term scalability.
