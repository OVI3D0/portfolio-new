My Projects

Spotify Clone

Made with:

Spotify clone built using the spotify API, and React.js.
The user is able to log in, and if there is music playing from their spotify account, the app will listen and display the currently playing song in the player at the bottom. The user is also able to see their created playlists on the side, and display the playlist by clicking on it.

Notes App

Made with:

Notes app built using React.js.
The user is able to save, delete, and view different notes using a markdown editor. The notes are saved in localstorage, and the user is able to see their notes even after closing and reopening the app. Users are also able to change between light and dark mode.

Finance Tracker

Made with:

Finance Tracker App built using React & Firebase as a part of NetNinja's React+Firebase course on udemy. The user is able to log in using firebase authentication, and store transactions. The app then saves these transactions for each user independently, and they are fetched from firebase collections upon logging in.

Pomodoro Website

Made with:

Pomodoro Website made using HTML, CSS, and JavaScript. Users can begin a pomodoro timer by pressing the 'play' button next to the timer in the sidebar. Then, a timer begins to count down from 25 minutes, the standard time for a pomodoro block. When the timer runs out, a gentle alarm sound is played. In the meantime, users can choose between 9 different 'workspaces', which consist of relaxing videos the user can play to help them focus on their work.
